# Just Fucking Use Nuxt

[Visit the website](https://justfuckingusenuxt.com) · [Read the mission](https://github.com/Just-Fucking-Use-Vue-js/justfuckingusenuxt.com/issues/1)

## Setup

Install dependencies with the package manager pinned in `package.json`:

```bash
pnpm install
```

## Development Server

Start the development server at `http://localhost:3000`:

```bash
pnpm dev
```

## Quality checks

Run the same checks enforced in CI:

```bash
pnpm lint
pnpm typecheck
pnpm build
```
